## 省钱建议

1. **eFoil 团购**：6 人一起谈，单价能砍到 2,500 THB / 30min
2. **按摩选街边正规店**：连锁贵 50%，街边正规 90min 油按 ~600 THB
3. **餐饮**：早餐别墅自做（如别墅有厨房）→ 午餐路边摊 → 晚餐稍好
4. **机票**：提前 2 个月订 / 转机便宜 30-50%
