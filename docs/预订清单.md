# 预订清单

> 建议出发前至少 3 天完成所有预订。Kong Hyrox 和 Pum Thai 需要电话/邮件确认开课，比订其他更紧急。

## 必订项

### 1. 别墅（4 晚 · 5BR 私泳池 · 50-70k THB/晚）

- **目标区域**：Rawai 北部 / Chalong 南部（Chalong Pier + Soi Ta-iad + Rawai Beach 三角内）
- **入住**：2026-06-18 / **退房**：2026-06-22 上午
- **房型要求**：
	- **5 BR / 5 卫**
	- 私人泳池（容纳 8 人 + 适合 10 人 tentative）
	- 厨房 + 大客厅（晚上聚餐 + 出证庆祝）
	- 容纳 **8 成人**（确认）+ 可加床（应对 +2 tentative）
- **推荐 villa 平台 / 社区**：
	- **Botanica Luxury Villas**（Layan / Bang Tao 高端）— 5BR 多，距 Chalong 略远 25min
	- **The Eva Phuket**（Rawai）— 5BR 整套 villa，距码头近
	- **Villa Sunpao / Villa Splash**（Layan）— Luxury Retreats 平台
	- **Villa Padma / Villa Sawarin**（Phuket 高端 5BR pool villa）
	- **Two-villa 备选**：Saiyuan / Phu Montra 拼订 2 栋 3BR（6 + 2 分栋）
- **搜索关键词**：
	- Airbnb：`Phuket 5 bedroom pool villa Rawai/Chalong`，过滤"整套 + 8 客人 + 私人泳池"
	- Booking：搜 "Rawai 5 bedroom pool villa" 筛"整套别墅"
	- 高端站：The Luxe Nomad / Luxury Retreats / Asia Villas
- **4 晚预算**：200,000-280,000 THB ≈ 40,000-56,000 RMB（人均 5,000-7,000 RMB）
- **状态**：⬜ 未订 ⚠️ **旺季 5BR 数量少，必须这周内发询价 + 多家平行问**
- **避坑**：
	- 别选 Patong / Kata — 每天潜店接送多 20-30 min
	- 别选 Cape Panwa / Cape Yamu — 高端但太远
	- 看清是否包早餐 + 司机服务 + 清洁费 + 押金（5BR 高端容易额外费）
	- 跟房东要 floor plan，避免某间房只有一张单人床住不下成年人
	- 加床政策：能不能临时加 2 张床应对 tentative
