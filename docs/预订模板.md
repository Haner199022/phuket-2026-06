## 8. 通用预订小贴士

### 邮件 / 私信 5 个原则
1. **主题清晰**：日期 + 人数 + 服务（"6 pax dive course Jun 18-22"）
2. **一次问完**：编号列问题，对方回复一次到位，省 3-4 轮拉扯
3. **预算线先暴露**：让对方明确这是认真客户，不是询价撒网
4. **留 WhatsApp**：泰国商家比邮件更爱用 WhatsApp / Line，回复快 5 倍
5. **保留所有截图**：付款凭证、确认邮件、退订政策 — 都存一个 `bookings/` 文件夹

### 付款方式优先级
- **信用卡（优先）**：Klook / Booking / Airbnb 平台单 → 有平台保护
- **WhatsApp 收款 / Wise**：直接联系的小商家
- **银行转账**：大额定金 / 别墅，留好 receipt
- **现金 THB**：街边按摩、tuk-tuk、零碎打赏

### 取消政策默认门槛
- 别墅：免费取消通常 30 天前 / 14 天前
- 潜店 / 健身：48h 前免费、24h 内 50% 罚
- 厨师课：24h 前免费、当天不退
- 接送：6h 前免费、3h 内 100% 罚
